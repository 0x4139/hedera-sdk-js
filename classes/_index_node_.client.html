setMaxQueryPayment

  • setMaxQueryPayment(maxPayment: Hbar): this

  • Inherited from BaseClient.setMaxQueryPayment

    • Defined in src/BaseClient.ts:179

    Set the max payment that can be automatically attached to a query.

    If this is not called then by default no payments will be made automatically for queries.

    If a query will cost more than this amount, a MaxQueryPaymentExceededError will be thrown from QueryBuilder.execute().

    This can be overridden for an individual query with query.setPaymentDefault(await query.requestCost()).

    Parameters

    • maxPayment: Hbar

      the maximum automatic payment for a query

    Returns this
